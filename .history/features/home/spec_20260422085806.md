## Goal 

implement `index.html`, and demonstrate the things I learned in user interfaces by creating a clean and minimalistic website to demonstrate my portfolio.

## Work
- Describe the UI/UX clearly and specifically: The website is a single column scrolling (mobile first) 
- What elements exist? What container elements exist?: A header with the name, navigation on the right for the home page and project details/work, the body with the main details for my home page, then a footer with a let's talk section with contacts, github, and copyright. The container elements are header, main, footer, each section has a .container div.
- How do users interact? What is the hover behavior? Nav links scroll you to the corresponding sections, the name returns you to the first section of the page if clicked, project rows are clickable and then take you to the project-details page when clicked, the email opens mailto, social links have to open in a new tab. The hover behavior: nav links shift colors when hovered over, project rows nudges slightly to the right and the title switches to italic, email switches to italic, social links shift from grey to black.
